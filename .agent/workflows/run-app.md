---
description: Start the CloudVault Secure Sharing application locally
---

To run the application for the first time or after updates:

1. Ensure dependencies are installed:
// turbo
```powershell
npm install
```

2. Start the development server:
// turbo
```powershell
npm run dev
```

3. Open your browser to the URL displayed in the terminal (usually `http://localhost:3000`).
